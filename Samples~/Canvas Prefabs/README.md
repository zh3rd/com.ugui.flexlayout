# Canvas Prefabs

这个 sample 包含 4 个 `1920x1080` 的 UGUI Canvas 预制体：

- `FlexValidationCoverCanvas`
  - 当前 README 使用的封面构图
- `FlexDashboardCanvas`
  - 深色数据看板风格
- `FlexEditorialCanvas`
  - 浅色编辑排版风格
- `FlexCommerceCanvas`
  - 商品展示与转化布局风格

这些预制体用于演示：

- `FlexLayout` 作为容器的基本组织方式
- `FlexNode` 对自身尺寸的控制
- `FlexItem` 对 `grow / shrink / basis` 的控制
- `TextMeshProUGUI` 在复杂卡片中的排布方式

导入后可以直接拖入场景，也可以作为新的界面起点继续修改。
